# Country Search

An autocomplete search for countries

## Pre-requisite

- Head over to [cg repository](https://github.com/trishtzy/cg) to start the server side using Docker

## Installation

```sh
npm install
```

## Build and Run

- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `npm run start`
- Run dev server: `npm run serve`
